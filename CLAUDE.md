# CLAUDE.md — Living Research Protocol Vault

This repository is both a **Git repo** and an **Obsidian vault** containing the Living Research Protocol (LRP) and related materials.

## Structure

- **Root:** Core LRP documents (PROTOCOL.md, PHILOSOPHY.md, README.md, etc.)
- **Design Notes:** `# LRP Design Note N: ...` files — design rationale
- **Guidance Notes:** `# LRP Guidance Note N: ...` files — practical guidance
- **case-studies/**: Real applications of LRP
- **development-trace/**: Process documentation, meta-work, tooling notes
- **Appendices/**: Extended frameworks (e.g., Truth Tester)

## Conventions

- All content is markdown (`.md`)
- File names often start with `#` — this is intentional for Obsidian heading display
- Use `[[wiki-links]]` for internal connections when creating new content
- Every claim should follow QITCS format (Question, Inputs, Transform, Confidence, Sunset)
- Development trace captures process, not just outcomes

## When editing this vault

- Place new LRP core documents in the root
- Place case studies in `case-studies/`
- Place meta/process notes in `development-trace/`
- Place extended frameworks in `Appendices/`
- Commit with clear messages describing what changed and why
- Do not modify the LICENSE file
