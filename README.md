## Quick Start

1. **Notice** what's alive (what won't let you sleep)
2. **Document** using QITCS format: Question, Inputs, Transform, Confidence, Sunset
3. **Fail()** - break it on purpose before committing
4. **Set confidence** as skeptic's survival odds (not your optimism)
5. **Set sunset** - when does this need review?

**Time investment:** 10 min to learn, 15 min per claim to document properly.
